# AfroTech 

## Description 

>>This project is  assigned to me by Microverse as a  final capstone project of module 1(HTML/CSS/JAVASCRIPT) from Microverse curriculum.

AfroTech festival is an online website for event and conference about the tech. 
AfroTech brings togheter all technology player such as startup , compagnies and individuals.
its also include conference where africa business leaders and pionner thinker discuss the industry most revelant issues. Afrotech is a place to share knowledge , build a growth network , exhibit lastest tech project and make business. 

**The website contains two pages:** 
- The home page and 
- The about page.

 The home page displays informations about the upcoming events and also the speakers that will host the conference. the main program and the advantage of the event are also highlighted.

 The about page explains more the concept of the event and contains pictures gallery of the previous event.
